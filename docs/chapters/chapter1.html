Getting started

Who is this guide for?

This guide is meant for both programmers and non-programmers, theory and practice will be kept separate so that you can skip the theory you already know if you want to.

If you are an already experienced programmer
I suggest you to skip all the teory in the first part (variables, flow control,...), but still if you never programmed in an object-oriented language you should read the more advanced theory, because it's language specific.

If you are new to programming
Read the whole guide and don't skip theory, it may be boring (not too boring don't worry), but without it you are not going anywhere; don't worry if you don't understand some concepts the first time, read it again and try doing some tests to understand how it works.

The tools

To program in C# you will need a text editor and a compiler, you can use whatever you want, but I highly recommend Visual Studio: it's both a text editor and a compiler, it has amazing features that help you writing and debugging the code and it's free. You can download it here

PRO TIP

If you are an hardcore programmer and command line fanatic check out roslyn. It's mantained by Microsoft and has some pretty nice features.

In the installation make sure to install all the c# packages
