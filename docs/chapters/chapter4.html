WHILE - PRACTICE

The while syntax as you could have guessed is:
while(condition){
code to execute
 }

PAY ATTENTION

If your while loop never ends the program will crash. So don't do things like
int x = 10;
while(x>0){
Console.WriteLine("I'm an endless loop");
 }

There are two more keywords that you should learn:
  • break: calling break will cause the software to exit the loop, without executing the following code nor checking the condition, it works with all the flow controls (if too). To call the break just write break;, e.g. 
    int count=0;
string userInput="";
while(userInput!="password"){
      count++;
      if(count>5){
            Console.WriteLine("You tried too many times, bye bye");
            break;
      }
      userInput=Console.ReadLine();
}
  • continue: this one only works with loops and when called ends the current cycle of the loop, it DOESN'T EXIT the loop, just skip the following code for this cycle. You can call it writing continue;
And for the while statement that's it, enjoy!

FOR - THEORY

The for can be thought of as a wrapper for a while loop
FOR LOOP EQUIVALENT WHILE LOOP
for (int i=0; i<=5; i++){ Console.WriteLine(i.ToString());
}		 int i=0;
while(i<=5){ Console.WriteLine(i.ToString());
i++;
}

FOR - PRACTICE

The syntax of the for loop is
for (declaration and assignment; condition; variable update){
Code to execute
}
